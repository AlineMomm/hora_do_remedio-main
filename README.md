# Hora do Remédio
Um aplicativo mobile para gerenciamento de medicamentos com foco em usabilidade para idosos
## Sobre o Projeto
O Hora do Remédio é um aplicativo desenvolvido em Flutter para ajudar usuários, especialmente idosos, a gerenciarem seus medicamentos de forma simples e eficiente. O app envia lembretes nos horários corretos e mantém um histórico organizado dos tratamentos.
## Funcionalidades
- Cadastro e Autenticação
- Gerenciamento de Medicamentos
- Lembretes Inteligentes
- Perfil do Usuário
- Interface Acessível
## Tecnologias Utilizadas
- Flutter
- Firebase
## Como Executar
- Clone o repositório
- flutter pub get
- flutter run
